Rentechify revolutionizes tech rentals with a seamless platform for on-demand access to premium gadgets and electronics. Our intelligent booking system with AI-powered support ensures a frictionless rental experience from selection to return.

## 🔄 Rental Workflow

```mermaid
sequenceDiagram
    participant User
    participant App as Rentechify App
    participant AI as AI System
    participant Agent
    
    User->>App: Browse available gadgets
    App->>User: Display options & pricing
    User->>App: Select gadget & timeframe
    App->>AI: Verify availability & eligibility
    AI->>App: Confirmation & payment request
    User->>App: Complete payment
    App->>Agent: Dispatch rental request
    Agent->>User: Deliver equipment
    Note over User,Agent: Rental Period
    User->>App: Initiate return process
    App->>AI: Run condition assessment
    AI->>App: Generate return report
    App->>User: Process deposit refund
```

## 🚀 Getting Started

```bash
# Clone repository
git clone https://github.com/codegeeksteam/RentifyTech

# Install dependencies
cd rentechify
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
